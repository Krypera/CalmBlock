Browser Notes

  • Support tiers are certification targets until a release matrix is filled and published for a tagged release.
  • Chromium certification uses one packaged chrome build for Chrome, Edge, Brave, Vivaldi, Opera, Arc, and Yandex.
  • Firefox-family certification uses one packaged firefox build, with Firefox as the reference target plus LibreWolf, Waterfox, Floorp, and Zen.
  • Orion is gated and stays non-official until it passes two consecutive release cycles with recorded validation.
  • Safari is a separate workstream with its own Apple packaging and review lane requirements.
  • When live counters are unavailable, blocking still works and UI shows fallback state clearly.

Trust Notes

  • Permission rationale and known limitations are documented in repository README.
  • No telemetry, no analytics, no remote code, no account sync.
  • What is not blocked is explicitly listed to avoid over-claiming protection scope.
