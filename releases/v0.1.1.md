# CalmBlock v0.1.1

CalmBlock `v0.1.1` is the current beta-ready open-source baseline.

## Highlights

- Privacy-first browser extension with no telemetry, no analytics, no remote code, and no accounts.
- Local packaged rules for ads, trackers, annoyances, and strict privacy mode.
- Improved release pipeline with package validation, browser support documentation, and clearer regression coverage.
- Stable direct-download assets for README install links:
  - `calmblock-chrome-latest.zip`
  - `calmblock-firefox-latest.zip`

## Downloads

- Chromium-family package: `calmblock-chrome-v0.1.1.zip`
- Chromium-family stable alias: `calmblock-chrome-latest.zip`
- Firefox-family package: `calmblock-firefox-v0.1.1.zip`
- Firefox-family stable alias: `calmblock-firefox-latest.zip`

## Notes

- Chromium-family browsers use the same `chrome` artifact: Chrome, Edge, Brave, Vivaldi, Opera, Arc, Yandex.
- Firefox-family browsers use the same `firefox` artifact: Firefox, LibreWolf, Waterfox, Floorp, Zen.
- Safari and Orion remain documented lanes, but they are not shipped as first-class release artifacts here.
